AngularJS

AngularJS is an obsolete version of Angular.

The CHT UI framework was upgraded from the deprecated AngularJS v1.6 to Angular in the version 3.11. The upgrade drastically reduces the memory used on the phone. It makes development of the Core Framework easier and more reliable and also makes it easier to keep on the latest version of Angular which means we can keep up to date and on a supported version.

Currently, it is only used in the CHT admin app. Deprecated: rewrite to Angular.
